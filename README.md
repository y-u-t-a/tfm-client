# tfm-client

[TOKYO FM Podcast](https://www.tfm.co.jp/podcast/) Client.

## Setup

Make sure to install dependencies:

```bash
vp install
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
vp run dev
```

## Testing

```bash
vp test
```

## Preview

Locally preview production build:

```bash
vp run start
```
