# tfm-client

[TOKYO FM Podcast](https://www.tfm.co.jp/podcast/) Client.

## Setup

Make sure to install dependencies:

```bash
npm install
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
npm run dev
```

## Testing

```bash
npm run test
```

## Preview

Locally preview production build:

```bash
npm run start
```
