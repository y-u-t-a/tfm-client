# tfm-client

[TOKYO FM Podcast](https://www.tfm.co.jp/podcast/) Client.

## Setup

Make sure to install dependencies:

```bash
bun install
```

## Development Server

Start the development server on `http://localhost:3000`:

```bash
bun dev
```

## Testing

```bash
bun test
```

## Preview

Locally preview production build:

```bash
bun start
```
